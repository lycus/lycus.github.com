Projects A list of projects managed by the Lycus Foundation.

MCI (Docs)

The MCI (Managed Compiler Infrastructure) is a modern and intuitive compiler infrastructure written in the D programming language. It provides a simple, extensible API specialized for modern-day programming language designs. It is primarily intended for so-called "managed" languages, which usually run in a virtual machine and typically use a garbage collector for memory management.

ExVM (Docs)

ExVM (Extensible Virtual Machine) is a JIT-compiling virtual machine built on top of the MCI (Managed Compiler Infrastructure). It is a much more high-level virtual machine, defining an ABI (application binary interface) for many common compiler features such as name mangling, static/TLS data initialization, runtime type information and reflection, reified generics, etc.

Cursive (Docs)

A functional-imperative language primarily used as a proof of concept for implementing a language on top of the MCI. It has a simple C-like syntax, with the addition of features such as classes/interfaces (OOP), generics, lambda functions with closures, (limited) type inference, and garbage collection.

libffi-d

Binding to the libffi library for the D programming language.

libgc-d

Binding to the libgc library for the D programming language.

Pegged

We maintain a fork of the Pegged parser generator where we develop various patches and fixes to maintain support for the various operating systems, compilers, and architectures that we support in our other projects. Additionally, our fork contains a proper Waf build system that integrates with the rest of our ecosystem.
